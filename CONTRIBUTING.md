# Contribuindo

Obrigado por considerar contribuir com o Vibes!

## Como começar
1. Faça um fork do repositório.
2. Crie uma branch com um nome descritivo.
3. Faça alterações pequenas e focadas.
4. Abra um Pull Request com descrição clara.

## Padrões de qualidade
- Rode `npm run lint` e `npm run test` antes do PR.
- Mantenha o foco em melhorias incrementais.
- Documente mudanças relevantes no README ou docs.

## Commits
Use mensagens curtas e objetivas.
