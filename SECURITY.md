# Política de Segurança

## Reportando vulnerabilidades
Envie detalhes para security@exemplo.com. Inclua:
- Passos para reprodução
- Impacto observado
- Versão/commit afetado

## SLA de resposta
Retorno inicial em até 7 dias úteis.

## Divulgação responsável
Solicitamos divulgação responsável para proteger usuários.
